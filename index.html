My jQuery Basic Assignment

A wireframe that shows jQuery methods

.addClass

Click the button to add class "red" to this paragraph.

.after

Click the button to insert content, specified by the parameter, after each element in the set of matched elements

.append

Click the button to insert content, specified by the parameter, to the end of each element in the set of matched elements

.attr

Click the button to get the value of an attribute for the first element in the set of matched elements or set one or more attributes for every matched element (See in Console)

.before

Click the button to insert content, specified by the parameter, before each element in the set of matched elements

.click

Click the button to see how .click binds an event handler to the "click" JavaScript event, or trigger that event on an element

.data

Click the button to store arbitrary data associated with the matched elements or return the value of the named data store for the first element in the set of matched elements

.fadeIn

Click the button to display the matched elements by fading them to opaque

.fadeOut

Click the button to hide the matched elements by fading them to transparent

.hide

Click the button to hide the matched elements

.html

Click the button to get the HTML contents of the first element in the set of matched elements or set the HTML contents of every matched element

.show

Click the button to display the matched elements

.slideDown

Click the button to display the matched elements in a sliding motion

.slideToggle

Click the button to display or hide the matched elements in a sliding motion

.slideUp

Click the button to display the matched elements with a sliding motion

.text

Click the button to get the combined text contents of each element in the set of matched elements, including their descendants, or set the text contents of the matched elements

.toggle

Click the button to display or hide the matched elements

.val

Click the button to get the current value of the first element in the set of matched elements or set the value of every matched element
